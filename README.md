# Simple Static Express API

Simple static API, based on JSON files stored on the server.

## Installation

Use the package manager [npm](https://npmjs.com/) to install the application.

```bash
npm install
```

You must manually create a **.env** file at the root of the project folder.  Add the following to the file :

```bash
PORT=3000
```

## Usage

Run the following command to run the application in development.

```bash
npm run dev
```

The application will run [nodemon](https://www.npmjs.com/package/nodemon) in development.
